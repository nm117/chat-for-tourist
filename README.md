# Features
 - Basic authentication with Firebase  
 - Chats - Read/Unread receipts, Pop-up user information, Create/Read/Delete  
 - Rooms - Pop-up room information, Roomlist for each user, Create/Read/  Update
 - Mypage - User can update user information includes avatar, CRUD  
 - Responsive Web Design, real-time updates for all pages  

# Usage
```
git clone https://github.com/nm117/chat-for-tourist.git  
cd chat-for-tourist

# Compiles and hot-reloads for development  
npm run serve

# Compiles and minifies for production  
npm run build
```
